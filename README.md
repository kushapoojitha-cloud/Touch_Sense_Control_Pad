✋ Touch Sense Control Pad

📌 Abstract

The **Touch Sense Control Pad** is an embedded system project that uses touch-sensitive inputs to control devices or trigger actions.
Instead of traditional mechanical switches, the system detects human touch using capacitive sensing and converts it into digital signals for processing. 
This project demonstrates modern, user-friendly input methods widely used in smart devices and automation systems.

🎯 Problem Statement

Conventional mechanical switches are prone to wear and tear, require physical force, and lack modern user experience. In contrast, touch-based systems provide 
a more reliable, efficient, and intuitive way of interaction.
This project aims to design a **touch-based control system** that replaces physical switches with touch sensors, improving durability and usability.

🎯 Objectives

* To detect touch input using capacitive sensing
* To process touch signals using a microcontroller
* To control output devices based on user interaction
* To implement a modern and efficient control interface
